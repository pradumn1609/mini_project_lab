# Student Result Management System

## Run locally

1. Install dependencies: `npm install`
2. Start json-server: `npm run json-server` (runs at http://localhost:4000)
3. Start dev server: `npm run dev` (runs at http://localhost:5173)

API base: http://localhost:4000

Routes:
- GET /students
- GET /sections
- GET /results

CRUD supported by json-server.
